1 Git and GitHub

Git is a free an open source distributed version control system. It was originally developed by Linus Tovvalds to manage changes to the Linux kernel.
GitHub is an online service that uses Git to provide code repositories. GitHub has added functionality in comparison, and is one of the most popular code repository systems worldwide.
In other words Github is a platform that helps people solve different problems by building software together.
